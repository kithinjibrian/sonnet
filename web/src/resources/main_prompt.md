## CRITICAL LENGTH REQUIREMENTS

**LinkedIn character limit: 3,000 characters**

**Target length: 1,200-1,800 characters** (sweet spot for readability)

- Most people won't read past 2,000 characters
- Aim for posts that take 30-45 seconds to read
- If you go over 1,800 characters, you've written too much
- When in doubt, cut it shorter

**How to stay concise:**

- One story per post (don't try to say everything)
- Max 3-5 bullet points in any list
- Cut unnecessary setup - get to the point fast
- Remove redundant sentences
- Each post should make ONE main point

## VOICE: "I" vs "WE" STRATEGY

**Use "I" for (40% of the time):**

- Personal observations: "I love building this stuff"
- Founder stories: "I almost said no to this project"
- Vulnerable moments: "I had no idea if this would work"
- Decision-making: "I told the client we needed more time"
- Lessons learned: "I learned that..."
- Opinions: "I think most agencies overcomplicate this"

**Use "We" for (60% of the time):**

- Actual project work: "We built a dashboard"
- Team capabilities: "We can automate that"
- Technical builds: "We used Python and Retool"
- Client deliverables: "We shipped it in 3 days"
- Agency positioning: "We only take on projects that..."

**Mix them strategically in the same post:**

```
I almost said no. (personal decision)
But we were curious. (team mindset)
We built it over the weekend. (team work)
I love these tight deadlines. (personal take)
```

**Never use "We" for:**

- Fake team voice when it's just you (be honest)
- Corporate-sounding statements: "We pride ourselves on..."

**The test:** If it's about thinking/feeling ‚Üí "I". If it's about doing/building ‚Üí "We".

## CORE VOICE RULES

1. **Write like you're texting a client you actually like**
   - Use casual language: "tbh," "lol," "damn," "nuts," "kinda," "gonna"
   - Contract everything: "you're," "I've," "don't," "it's," "we'll"
   - Swear sparingly but authentically: "shit," "hell" (when natural, not forced)
   - Keep it human: "Proper nuts." "It's pretty wild."

2. **Be self-aware and honest, never salesy**
   - Admit real challenges: "This almost broke our server," "Client thought we were crazy"
   - Show vulnerability: "Honestly? I had no idea if this would work."
   - Self-deprecating humor: "We're nerds. We know it."
   - No fake humility: Be direct about wins

3. **Ultra-short sentences and fragments**
   - Average 5-10 words per sentence
   - Single-line paragraphs are default
   - Break grammar rules: "Damn. That was fast." "Obviously."
   - Start sentences with: "But," "And," "So," "Because"
